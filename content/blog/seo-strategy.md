SEO（搜索引擎优化）是一种通过优化内容和技术手段，提升网站在搜索引擎（如谷歌）中排名的策略。它的核心目标是让用户在搜索特定关键词时，更容易找到你的页面、网站或视频等内容。

SEO 是一个`长期`性价比极高的获客渠道

## Google 搜索的工作原理

1. **爬取网页（Crawling）**：
    - **Scheduler**：调度器决定哪些网页需要被爬取。它会根据网页的重要性、更新频率等因素来安排爬取任务。
    - **Awwler**：这是Google的爬虫工具，负责访问网页并收集数据。它会遵循网页上的链接，发现新的内容。
        
2. **预处理（Preprocessing）**：
    - **NLP（自然语言处理）**：对抓取的内容进行自然语言处理，以理解文本的含义和结构。
    - **个性化（Personalisation）**：根据用户的搜索历史和偏好，对内容进行个性化处理。
        
3. **索引（Indexing）**：
    - **Parser**：解析器将网页内容分解为可索引的部分，如文本、图像等。
    - **Semantic Indexing**：语义索引通过理解上下文来创建索引，使得搜索引擎能够更好地理解内容的相关性。
    - **Indexed Pages**：经过解析和语义索引的网页被存储在索引库中，以便快速检索。
        
4. **查询处理（Query Processing）**：
    - **Search Query Input**：用户通过文本、语音或视觉输入搜索查询。
    - **Query**：搜索引擎处理用户的查询，理解其意图和需求。
        
5. **排名（Ranking）**：
    - **AI-driven Ranking**：使用人工智能技术（如BERT、MUMA、EEAT）对网页进行排名。这些技术帮助搜索引擎理解查询的复杂性和网页内容的相关性。
    - **Ranked Pages**：根据相关性、权威性和其他因素，网页被排序并呈现给用户。
        
6. **结果呈现（Result Presentation）**：
    - **AI Summaries + Links**：搜索引擎提供AI生成的摘要和相关链接，帮助用户快速找到所需信息。

`Crawl` -> `Index` -> `Rank` -> Convert
- Crawl   确保内容能够被谷歌爬虫访问
- Index   提升内容品质->被收录
- Rank    优化相关性->排名高
- Convert 符合用户预期->提升转化

## SEO优化的五个模块

- **关键词调研**：
    - 通过研究`用户搜索`的关键词，确定目标关键词和长尾关键词，帮助优化内容和排名策略，从而吸引相关流量并提高转化率。
        
- **内容优化**：
    - 创建高质量、`关键词相关`且有价值的内容，确保内容被正确索引，并在搜索结果中获得较高排名。优质内容还能提高用户参与度和转化率。
        
- **On-page和结构优化**：
    - 优化页面元素（如标题标签、元描述、URL结构等）和网站`页面结构`，确保搜索引擎爬虫能够有效抓取和索引内容，同时提高页面的相关性和排名。
        
- **外部链接建设**：
    - 通过获取高质量的外部链接（`backlinks`），提高网站的权威性和可信度，从而提升在搜索引擎结果中的排名。
        
- **技术SEO**：
    - 解决网站的技术问题，如网站速度、移动友好性、结构化数据、XML站点地图等，确保搜索引擎能够顺利抓取和索引网站内容。

- 不同的国家搜索结果不同
- 本地搜索结果
- Organic result 未通过付费广告获得的搜索结果。
- Sponsored 付费广告
- People also ask (PAA)

## 影响谷歌排名的8大因素

1. **页面内容与搜索词的相关性**
	- relevance
2. **页面关键词布局**
	- 关键词出现的程度 位置 频率
3. **页面权重（外链）**
	- 权威程度
4. **用户交互信号**
	- 点击时间 跳转页面 停留时间 跳出率
5. **内容更新频度**
6. **站点加载速度**
7. **站点结构特性**
8. **站点品牌信任度**


## 关键词调研

用户搜索 -> 关键词 <- 页面内容

### 明确目标受众需求
`KDROI`（Keyword Difficulty Return on Investment）优化难度回报率是一个用于评估关键词价值和优化策略的指标。它结合了关键词的搜索量、每次点击费用（`CPC`）和关键词难度（`KD`），帮助你判断一个关键词是否值得投入资源进行优化。
```
KDROI = Volume×CPC​ / kd
```

通过计算每一个关键词的 kdroi ，再对关键词按照 kdroi 从高到低排序，就得到了最值得做的关键词列表。
下载csv或者xlsx文件，在电脑里打开。保留 Keyword、Volume、KD、CPC这四列，把其它列都删除。  
然后新增一列，输入公式 =B2 * D2 / C2 ，拉一下这一列就计算出来了优化难度回报率。之后按照这一列从高到低排序，就得到了一个新表格。
### 寻找合适的关键词
- 从用户的角度出发
- Keyword Planner - Google Ads
- keywordtool.io
- ryrob.com/keyword-tool
- `KD`值为0到29，再设置`CPC`>=0.1美元，再按照搜索量从高到低排序，就得到了一批有一定搜索量且优化难度低且有一定价值的关键词。
	- KD (Keyword Difficulty)：关键词难度
		- 范围通常是0-100
		- 数值越高，说明竞争越激烈，越难排名

	- CPC (Cost Per Click)：每次点击成本
		- 在付费广告中每点击一次的费用
		- 单位通常是美元或当地货币
		- 反映关键词的商业价值：
		    - CPC高 = 竞争激烈，变现价值高
		    - CPC低 = 竞争较小，商业价值可能较低
- 使用[`Google Trends`](https://trends.google.com/trends/explore?geo=US&hl=en)检查**季节词，不稳定的流量**。
- 避免**谷歌直达**，我们搜索时，谷歌会直接在搜索结果最前面就把答案直接显示出来。
